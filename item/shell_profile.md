`ps -p $$` 명령어는 **현재 셸 프로세스의 정보를 출력**합니다.

- `ps`는 프로세스 상태를 보여주는 명령어입니다.
- `-p` 옵션은 특정 PID(프로세스 ID)를 지정합니다.
- `$$`는 **현재 셸의 프로세스 ID**를 의미합니다.
  $ : 변수 값
  $$ : 셸의 프로세스 ID

즉,  
`ps -p $$`를 입력하면  
내가 사용 중인 셸(터미널)의 프로세스 정보(프로세스 ID, TTY, 실행 시간, 명령어 이름 등)를 볼 수 있습니다.

**OSCP 실전 활용**

- 현재 사용 중인 셸 종류(예: bash, ash, sh 등)를 확인할 때 사용합니다.
- AppArmor, SELinux 등 보안 모듈이 특정 셸에 제한을 걸었는지 확인할 때도 활용합니다.
- 권한 상승 후, root 셸로 바뀌었는지 확인할 때도 사용합니다.

**예시 출력**

```
PID   TTY      TIME     CMD
1376  pts/0    00:00:00 ash
```

여기서 `CMD`가 현재 셸 이름입니다.

**요약**

- 내 셸의 프로세스 정보를 확인하는 명령어
- 셸 종류, PID, TTY 등 확인 가능
- 권한 상승, 보안 모듈 우회 등 실전에서 자주 사용
